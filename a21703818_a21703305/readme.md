#**O que aprendemos com este trabalho?**

Com este trabalho para a cadeira de Introdução à Computação, aprendemos e percebemos quando e onde se usa o "switch" , "for" , as funções e as passagens tanto por referencia como por valor.
Concluímos que a linguagem C é uma linguagem fundamental e extensa mas foi graças a este trabalho que conseguimos entender melhor a matéria dada em aula.
